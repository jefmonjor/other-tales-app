# Contexto de Negocio: Other Tales (Frontend Scope)

## Visión
App móvil para escritores. Foco en escritura offline, organización limpia y sincronización.

## Fases de Implementación (UI Scope)

### Fase 0: Design System Setup
- Definición de Colores, Tipografía y Espaciado.
- Creación de componentes base (Botones, Inputs, Tarjetas).

### Fase 1: Identity & Onboarding
- **Splash Screen:** Carga de configuración local/check auth.
- **Login/Register:** Formularios con validación.
- **Social Login:** Botones visuales (Mock action).

### Fase 2: Dashboard (Projects)
- **Home:** Grid de novelas. Placeholder para portadas.
- **Create Project:** Modal o Pantalla nueva.
